## ABOUT : 💡

**Explorer** is a search engine which is created using **Reactjs** , **Nextjs** , **Tailwind CSS** and hosted at Vercel.
Explorer supports user's favorite dark mode. ☀️ 🌙 
Explorer UI is partially inspried from google.
